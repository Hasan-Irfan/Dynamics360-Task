# Featured Events Page

## Technologies Used
- React
- Bootstrap 5
- JavaScript
- HTML5 & CSS3

## Features
- Responsive design
- Featured events section
- Search bar to filter events
- Dynamic event loading using dummy JSON

## How to Run
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
